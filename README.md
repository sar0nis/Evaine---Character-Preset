— READ ME —

INSTALLATION
1. Extract the .json file from the archive.
2. Place it in:
   Skyrim Special Edition\Data\SKSE\Plugins\CharGen\Presets

IN-GAME USE
1. Launch the game and open RaceMenu.
2. Set race to Breton and gender to Female.
3. Go to the Presets tab.
4. Press F9 and select the "GtS - Evaine" preset.
5. Confirm.

Done.
